Inisialisasi workspace monorepo untuk proyek "Kas Kita" (Aplikasi Pencatatan Uang Kas Kelas berbasis PWA).

Spesifikasi teknis utama:
- Tech Stack: React PWA (Frontend) & Java Spring Boot (Backend).
- Integrasi Layanan: Firebase (Auth, Firestore, Storage, Cloud Messaging) di kedua sisi.

Tugas:
1. Buat struktur folder proyek monorepo dengan memisahkan direktori `backend` dan `frontend`.
2. Di folder `backend` (Spring Boot Maven), buat boilerplate layered architecture dengan root package `com.kaskita` (terdiri dari sub-package: config, controller, service, repository, model, exception, util).
3. Di folder `frontend` (React), buat boilerplate struktur React PWA di dalam folder `src/` (terdiri dari sub-folder: components, pages, hooks, services, context, utils, assets).
4. Buat file konfigurasi dasar:
   - `pom.xml` untuk backend dengan dependency Spring Web, Spring Security, database/Firestore, dan Firebase Admin SDK.
   - `package.json` untuk frontend dengan dependency React PWA standard dan Firebase Client SDK.

Catatan: Fokus hanya pada inisialisasi struktur direktori dan konfigurasi dependency dasar. Belum perlu menulis logika bisnis.
